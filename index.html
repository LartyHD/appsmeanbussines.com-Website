Lars Artmann

Software-Engineering, Technology-Consulting, Software Development

Passionate about technology, I keep myself busy with programming (mainly Kotlin & Java), cloud-native applications (e.g. Google Cloud), Software Architecture… At age 11, I found that my favorite game, Minecraft, had endless opportunities to code. So I started to build up my own Minecraft Server. On this project, I learned a lot about creating backend software in Java and Kotlin with tools like Git, Gradle and IntelliJ IDEA. With the help of a great mentor, this project advanced at a fast pace to more than 7,000 players per day. It was entirely managed by myself and a developer friend. At age 15, I started a company, Astride UG registered in Frankfurt, Germany. Its focus is on creating software architecture that makes an impact on the world. Innovative technologies are my strengths. Being able to learn new technology at a fast pace has allowed me to advance in fields like 3D printing, VR, AI, vibration detection, blockchain and many others. I am convinced that technology can improve the lives of many people around the world!
